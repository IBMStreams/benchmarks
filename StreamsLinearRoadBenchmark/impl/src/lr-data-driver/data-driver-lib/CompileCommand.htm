Files

  • LRDataProvider.h: header file of linear road data driver
  • LRDataProvider.cpp: linear road data driver
  • Tuple.cpp: structure of a linear road tuple (a line)
  • MemTuples.cpp: buffer of linear road tuples

Compile to a library file

  • Creating object files 
    g++ -g -O -c LRDataProvider.cpp MemTuples.cpp Tuple.cpp
  • Linking libraries 
    ar cru libLRDataProvider.a LRDataProvider.o MemTuples.o Tuple.o
				
    ranlib libLRDataProvider.a
